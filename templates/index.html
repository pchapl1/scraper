Welcome to ScrapeyMcScraperton

You can search for the following things from Stack Overflow...

  • Questions
  • Who asked the question
  • the tags for the question
  • the date the person asked the question


You can also use the following examples in the address bar to access more specific info..

  • /posts/ i.e. posts/3 returns three posts. i.e. posts/ returns all posts
  • /date/ i.e. /date/10-03-2021 returns posts from this date
